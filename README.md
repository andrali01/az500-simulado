# AZ-500 Simulado — Microsoft Azure Security Technologies

Aplicativo de simulado para a certificação AZ-500, com 276 questões oficiais.

## Funcionalidades
- 276 questões do exame AZ-500
- Modo Estudo (feedback imediato)
- Modo Exame (simulação real)
- Modo Aleatório
- Timer, score, revisão completa

## Deploy
Azure Static Web Apps via GitHub Actions.
